Esta aplicação foi criada a partir das seguintes tecnologias:

  • Angular, Bootstrap e jQuery para o frontend.
  • Web API (.NET Core) para o backend.
  • RabbitMQ para troca de mensagens.
  • Console App (.NET Core) para construção do robô de leitura da fila e inclusão das informações no Couchbase e SQL Server.
  • Couchbase e SQL Server para armazenamento dos dados (NoSQL e RDBMS).
  • Microsoft Azure para hospedagem da aplicação: http://viajanetwebapp.azurewebsitesn.net

URL da aplicação:

  • https://viajanettestern.azurewebsites.net

URL da API para ler os dados do Couchbase:

  • https://viajanettestern.azurewebsites.net/api/coleta?ip=&pagina=

Exemplos de chamada:
